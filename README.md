# Pinpoint Docker

Dockerfile for [Pinpoint](https://github.com/naver/pinpoint).

## Installation

``` sh
docker pull yous/pinpoint
```

## Usage

``` sh
docker run -i -t --rm yous/pinpoint bash
```
